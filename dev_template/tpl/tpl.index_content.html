安装步骤：

下载nodeJS：下载，并安装

安装Grunt：GruntJS。安装的过程中先安装npm install -g grunt。再安装npm install -g grunt-cli。

基于KS-Grunt的项目基本构建包

下载：KS-Grunt

基本构建包包含：css、js合并，混淆压缩，图片优化、HTML删除注释。
